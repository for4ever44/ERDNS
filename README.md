# ERDNS
 

## Dependencies
* Python 3.9
* PyTorch 1.8
* Numpy 1.20
* Scipy 1.6



## Usage

To run conventional models and ConvE model.
bash run.sh train TransE YAGO3-10 1 1 1024 1024 500 24.0 1.0 0.0002 200000 4 -s 0.5

To run CompGCN model.

## Data
We provide three datasets: FB15K237, WN18RR and YAGO3-10.
